Cursor.js
=========

安卓电视端光标（焦点框）移动插件，依赖于jQuery或Zepto。

示例
=========
```html
<div class="S-table" id="demo1" data-nth="0" data-focus=",,,">
	<div class="S-tr">
		<div class="S-td"></div>
		<div class="S-td"></div>
		<div class="S-td"></div>
		<div class="S-td"></div>
		<div class="S-td"></div>
	</div>
	<div class="S-tr">
		<div class="S-td"></div>
		<div class="S-td"></div>
		<div class="S-td"></div>
		<div class="S-td"></div>
		<div class="S-td"></div>
	</div>
</div>
<script>
SNTV.Cursor();
</script>
```
